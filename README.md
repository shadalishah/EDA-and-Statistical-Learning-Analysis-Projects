# 📊 Data Analysis & Statistical Learning Projects in Python

🚀 *End-to-end data analysis projects demonstrating Exploratory Data Analysis (EDA), visualization, and statistical reasoning on real-world datasets.*

---

## 🔍 Overview

This repository showcases **hands-on data analysis and statistical learning projects** using real-world datasets. The work is inspired by *An Introduction to Statistical Learning with Applications in Python* and focuses on developing practical skills required for roles in:

- Data Analysis  
- Business Intelligence  
- Economic Research  

The workflow includes:
- Data cleaning and preprocessing  
- Exploratory Data Analysis (EDA)  
- Data visualization  
- Statistical interpretation and insight generation  

---

## 💼 Key Skills Demonstrated

- Data Cleaning & Preprocessing  
- Exploratory Data Analysis (EDA)  
- Data Visualization (scatterplots, histograms, boxplots)  
- Feature Engineering  
- Statistical Analysis & Interpretation  
- Python (pandas, numpy, matplotlib, ISLP)  

---

## 📂 Datasets & Analysis

### 🎓 College Dataset
- Analyzed 777 US universities  
- Explored relationships between tuition, enrollment, and student quality  
- Created an **"Elite"** category based on academic performance  

### 🚗 Auto MPG Dataset
- Investigated factors affecting fuel efficiency  
- Identified strong relationships between MPG, weight, horsepower, and displacement  

### 🏙️ Boston Housing Dataset
- Analyzed housing prices and crime rates  
- Identified socioeconomic patterns and key predictors of housing values  

---

## 📈 Key Insights

- Strong correlations exist between application, acceptance, and enrollment rates  
- Vehicle weight and horsepower are major predictors of fuel efficiency  
- Higher-income areas tend to have lower crime rates and higher property values
