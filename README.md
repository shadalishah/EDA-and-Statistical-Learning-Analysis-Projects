📊 Data Analysis & Statistical Learning Projects in Python

🚀 End-to-end data analysis projects demonstrating EDA, visualization, and statistical reasoning on real-world datasets.

🔍 Overview

This repository demonstrates Exploratory Data Analysis (EDA) and statistical learning techniques applied to real-world datasets. The project is inspired by An Introduction to Statistical Learning and focuses on developing practical data analysis and problem-solving skills using Python.

The workflow includes data cleaning, preprocessing, visualization, and interpretation of results to extract meaningful insights — similar to real-world data analyst tasks.

💼 Key Skills Demonstrated
Data Cleaning & Preprocessing
Exploratory Data Analysis (EDA)
Data Visualization (scatterplots, histograms, boxplots)
Feature Engineering
Statistical Analysis & Interpretation
Python (pandas, numpy, matplotlib, ISLP)
📂 Datasets & Analysis
🎓 College Dataset
Analyzed 777 US universities
Identified relationships between tuition, enrollment, and student quality
Created “Elite” category based on academic performance
🚗 Auto MPG Dataset
Explored factors affecting fuel efficiency
Found strong relationships between MPG, weight, horsepower, and displacement
🏙️ Boston Housing Dataset
Analyzed housing prices and crime rates
Identified socioeconomic patterns and key predictors of housing value
📈 Key Insights
Strong correlations exist between application, acceptance, and enrollment rates
Vehicle weight and horsepower are key predictors of fuel efficiency
Higher-income areas tend to have lower crime rates and higher property values
